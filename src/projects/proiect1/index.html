Determinarea unui ciclu eulerian prin reuniune de cicluri

        

TERMINOLOGIE

Graf neorientat = o pereche ordonată de mulțimi , unde:

  •  este o mulțime finită și nevidă de elemente numite vârfuri sau noduri;
  •  este o mulțime finită de submulțimi cu două elemente din , numite muchii.

Muchii incidente = muchii care au o extremitate comună.

Grad = gradul unui nod din graf, este un număr natural ce reprezintă numărul de noduri adiacente cu                                    acesta.

Vârf izolat = un vârf cu gradul 0.

Ciclu = un lanț în care primul nod coincide cu ultimul.

Lanț eulerian= lanțul care traversează (m-1) muchii.

        Ciclu eulerian = ciclul care traversează toate muchiile grafului.

        Graf eulerian = graful care conține un ciclu eulerian.

        Teoremă : Un graf , care nu conține noduri izolate, este eulerian dacă și numai dacă este                conex și gradele tuturor nodurilor sunt numere pare.

        Graf conex = graful în care există lanț între oricare 2 noduri.

        

        PAȘII DE REZOLVARE

        2)Verific dacă graful este eulerian, folosind teorema ce presupune faptul că nu există noduri izolate,graful este conex și gradele tuturor nodurilor sunt pare. Dacă graful conține noduri izolate, algoritmul se poate aplica după înlăturarea acestora.

        Prin intermediul subprogramului conex apelez și subprogramul Dacă întâlnesc un nod nevizitat while-ul se oprește din execuție si se adaugă în stiva  nodul respectiv. Variabila p reprezintă numărul de elemente.

        

        Dacă am reușit să parcurgem toate nodurile și există astfel lanț între oricare 2 noduri, subprogramul conex returnează 1, altfel acesta returnează 0 și graful nu este conex.
