## Cerințe tehnice

- **Node.js**: Versiunea 14 sau mai recentă
- **npm**: Versiunea 6 sau mai recentă (npm vine împreună cu Node.js)

## Instalare

1. **Clonați repository-ul:**
```
git clone https://github.com/username/vite-project.git
cd intercode
```

2. **Instalați dependențele:**
```
npm install
```

## Rulare
```
npm run dev
```
